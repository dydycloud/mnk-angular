    Mnk-Angular

    Mnk-Angular is A Boilerplate App write in:

    • CofffeScript

    With:

    • Twitter Bootstrap dark, light and default theme
    • ui-bootsrap directives sets
    • Routing
    • Karma
    • Jasmine

    Install

    • git clone https://github.com/dydycloud/mnk-angular.git && cd mnk-angular
    • npm install && bower install

    Usage

    • Grunt tasks: grunt
    • Tests: ./node_modules/karma/bin/karma start
